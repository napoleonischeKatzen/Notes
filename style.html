The shortest string which contain all permutations of a set of n elements is what?\n Chapter I: The Structure of Permutations \n  
A
permutation is a finite sequence of elements of a set; the length of a permutation w is its number of elements and if a permutation has the length at least i then \(w(i)\) is its ith letter. A permutation of length n has the elements \([n]={1,2,...,n}\) and each element exist exactly once; therefore permutations are a special type of elements of the positive integer numbers \(\mathbb{P}\). Two permutations \(u,v\in\mathbb{P}^{n}\) which have the same length n and have positive integer elements are order-isomorphic if with all indices \(i,j\in[n]\) there is \(u(i)>u(j)\Longleftrightarrow v(i)>v(j)\).
